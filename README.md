Proyecto orientado a facilitar la identificación y gestión de piezas desde un dispositivo móvil.

- TypeScript
- SCSS
- HTML

## Requisitos
- Node.js (versión LTS recomendada)
- npm o yarn

## Instalación
1. Clonar el repositorio:
   git clone https://github.com/Magn01101111/vayo-mobile.git
2. Entrar al directorio:
   cd vayo-mobile
3. Instalar dependencias:
   npm install
   # o
   yarn install

> Nota: Revisa `package.json` para confirmar los scripts disponibles (`start`, `dev`, `build`, etc.).

## Ejecución (desarrollo)
Usualmente:
- npm run dev
- npm start

Para producción:
- npm run build

Si la app usa un framework móvil (Ionic, Capacitor, React Native, etc.), sigue los comandos específicos de ese framework para empaquetar y desplegar en emuladores/dispositivos.
